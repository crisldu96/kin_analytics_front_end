# Challenge 1

### DESCRIPTION 

This project was developed in React.js according to the instructions of the challenge 1, the main idea is to show my skills on design, knowledge and creativity. 

### REQUIREMENTS

Install Node.js

### HOW TO USE
1. Clone or download the project
2. `cd` the name of the project
3. execute `npm install`
4. Run the project with `npm start`
5. Go to `localhost:3000`
